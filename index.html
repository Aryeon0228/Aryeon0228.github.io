🌐 3D Gaussian Splatting (3DGS) 작업물

고해상도 3DGS 씬 재구성

Neural Radiance Fields (NeRF) 연구를 바탕으로 한 3D Gaussian Splatting 기법을 활용한 실시간 렌더링 가능한 3D 씬 재구성 작업입니다. 높은 품질의 포인트 클라우드 데이터로 사실적인 3D 환경을 구현했습니다.

파일 정보:
• 형식: PLY (Point Cloud)
• 크기: 449MB
• 포인트 수: 고밀도 포인트 클라우드
• 특징: 실시간 렌더링 최적화

📋 뷰어에서 확인하는 방법

  1. 아래 '파일 다운로드' 버튼으로 PLY 파일을 다운로드합니다
  2. ⚠️ 대용량 파일 주의:
    구글 드라이브에서 "바이러스를 검사할 수 없습니다" 경고가 나타날 수 있습니다.
    "다운로드 계속" 또는 "다운로드" 버튼을 클릭하여 진행하세요.
  3. 'Supersplat 에디터 열기' 버튼을 클릭하여 뷰어를 엽니다
  4. 에디터에서 다운로드한 PLY 파일을 업로드합니다
  5. 실시간으로 3D 씬을 탐색하고 다양한 각도에서 확인할 수 있습니다
📥 파일 다운로드 (449MB) 🚀 Supersplat 에디터 열기
3D Gaussian Splatting Point Cloud Real-time Rendering NeRF PLY Format
💡 대용량 파일이므로 다운로드에 시간이 걸릴 수 있습니다.
Wi-Fi 환경에서 다운로드하시는 것을 권장합니다.
