🌐 3D Gaussian Splatting (3DGS) 작업물

고해상도 3DGS 씬 재구성

Neural Radiance Fields (NeRF) 연구를 바탕으로 한 3D Gaussian Splatting 기법을 활용한 실시간 렌더링 가능한 3D 씬 재구성 작업입니다. 높은 품질의 포인트 클라우드 데이터로 사실적인 3D 환경을 구현했습니다.

파일 정보:
• 형식: PLY (Point Cloud)
• 크기: 449MB
• 포인트 수: 고밀도 포인트 클라우드
• 특징: 실시간 렌더링 최적화

📋 뷰어에서 확인하는 방법

  1. 아래 '파일 다운로드' 버튼으로 PLY 파일을 다운로드합니다
  2. ⚠️ 대용량 파일 주의:
    구글 드라이브에서 "바이러스를 검사할 수 없습니다" 경고가 나타날 수 있습니다.
    "다운로드 계속" 또는 "다운로드" 버튼을 클릭하여 진행하세요.
  3. 'Supersplat 에디터 열기' 버튼을 클릭하여 뷰어를 엽니다
  4. 에디터에서 다운로드한 PLY 파일을 업로드합니다
  5. 실시간으로 3D 씬을 탐색하고 다양한 각도에서 확인할 수 있습니다

🎨 Comfy UI & LoRA 모델 활용 사례

Stable Diffusion 기반의 고급 워크플로우 도구인 Comfy UI를 활용하여 LoRA(Low-Rank Adaptation) 모델을 적용한 이미지 생성 예시입니다. 복잡한 노드 기반 워크플로우를 통해 더욱 정교하고 일관된 스타일의 게임 아트워크를 제작할 수 있습니다.

LoRA 모델 특화 기능
  • 스타일 일관성: 특정 아트 스타일을 학습한 LoRA로 통일된 비주얼 구현
  • 워크플로우 최적화: 노드 기반 인터페이스로 복잡한 생성 과정 자동화
  • 실시간 조정: 파라미터 실시간 변경으로 빠른 이터레이션
  • 배치 처리: 여러 변형 이미지를 한 번에 생성하여 효율성 극대화

💡 실무 활용도: 게임 개발에서 컨셉 아트, 캐릭터 디자인, 환경 에셋 등의 초안을 빠르게 생성하여 아티스트의 작업 시간을 단축하고 더 많은 아이디어를 탐색할 수 있습니다.

✨ 아트 디렉션 & 스타일 가이드

색상, 명도, 채도를 활용한 화풍 가이드

아트팀 리더로서 아트워크의 일관된 색감과 화풍을 유지하기 위해, 색상(Hue), 명도(Value), 채도(Saturation) 이론을 기반으로 시각 자료를 활용하여 팀원들에게 가이드라인을 제시했습니다. 다양한 레퍼런스 이미지를 통해 특정 색감이나 화풍을 어떻게 분석하고 재현하는지에 대한 이론적, 실용적 접근법을 공유하여 아트워크의 퀄리티와 통일성을 높였습니다.

주요 내용:

  • 색상 이론 적용: 아트워크에 적용할 주된 색상 팔레트와 보조 색상 사용 가이드.
  • 명도 대비: 오브젝트의 입체감을 살리고 시선을 유도하는 명도(밝기) 조절 전략.
  • 채도 활용: 아트워크의 분위기와 감정을 조절하는 채도(색의 순도) 운용 방안.
  • 화풍 분석: 특정 화풍(예: 캐주얼, 실사, 반실사)의 시각적 특징을 HVS 기반으로 해부.
  • 실제 사례 분석: 유사한 색감/화풍의 게임 아트워크 분석을 통해 적용 방안 제시.

💻 Custom Shader Development

Unity에서 직접 쉐이더 코드를 작성하고 디버깅하는 실제 개발 과정입니다. 이론적 지식을 바탕으로 HLSL 코드를 구현하여 게임에 필요한 커스텀 렌더링 효과를 만들어냅니다.

컨셉에 맞춘 쉐이더 연구

HLSL 쉐이더 코드 작성

🛠️ 쉐이더 개발 역량
  • HLSL 코드 작성: 버텍스/프래그먼트 쉐이더를 직접 코딩하여 커스텀 효과 구현
  • 실시간 디버깅: Unity Inspector에서 속성값을 조정하며 실시간 결과 확인
  • 렌더링 파이프라인 이해: GPU 렌더링 과정과 최적화 기법 적용
  • Cross-Platform 호환성: 다양한 플랫폼에서 안정적으로 동작하는 쉐이더 제작

💡 개발 접근법: 시각적 결과물을 먼저 기획하고, 수학적 공식을 HLSL로 구현한 뒤, 최적화와 호환성을 고려하여 완성도 높은 쉐이더를 제작합니다.
